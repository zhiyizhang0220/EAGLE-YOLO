# Ethics and responsible use

The dataset was collected in agricultural field environments for maize seedling detection and robotic intra-row weeding research. The repository does not intentionally contain human-subject data.

Users should consider the following before reuse or deployment:

- Validate detection safety before operating robotic weeding actuators.
- Avoid using the model as the sole safety mechanism in field robots.
- Respect farm privacy and avoid unnecessary disclosure of precise farm-level location details.
- Cite the associated manuscript and this repository when using the data or model assets.
